# Robot Framework - Web and API testing

Xvfb installation:

    $ sudo apt-get install xvfb

Running tests: 

    # all tests
    $ robot *.robot
    
    # selected test
    $ robot test_ssh.robot
    $ robot test_web.robot
    $ robot test_sql.robot
